Stuff I use:

  • Here's the command you can use to get most of these utilities.

    You should probably have vim, some type of network manager and some other stuff but the arch user installation guide should tell you about that.

    yay -S mate mate-extra xrandr arandr thunar alacritty terminator nnn feh flameshot dunst copyq ttf-jetbrains-mono vscodium-bin rofi rofi-emoji peek mpv obs-studio spotify

  • Software/Configurations for Arch

    WARNING: My configurations take into account the fact that you are also using the other apps I use. All the important apps are listed below. Don't just copy paste my configs. Look at them and edit them to your liking before actually using them as your own. I am not responsible for any breakage of your system.

    • My dotfiles -> GitHub
    • AUR Helper/Wrapper -> yay AUR
    • Desktop Environment -> MATE
    • Display Manager -> xrandr with arandr

      arandr is the gui for xrandr

      Only really useful if you have more than one monitor

    • GUI File Explorer -> thunar
    • Shell -> bash
    • Terminal Emulator -> alacritty & terminator

      Alacritty doesn't play well with ssh so I use terminator for ssh-ing

    • Terminal File Explorer -> nnn
    • Terminal Image Viewer -> feh
    • Screenshot -> flameshot with dunst (dunst is the notification daemon that makes sure flameshot doesnt freeze up)
    • Clipboard Manager -> copyq
    • Monospace Font -> Jetbrains Mono Jetbrains Site 
      sudo pacman -S ttf-jetbrains-mono
    • Editor -> code (VSCode)
      • Theme -> One Dark Pro Theme
      • Sublime keybinds
    • Hotkeys/Keybinds -> sxhkd GitHub ArchWiki

      Uses KeySym to represent keys

    • App Launcher -> rofi GitHub ArchWiki
    • Emoji -> rofi-emoji GitHub AUR
    • Gif Recorder -> peek
    • Video Recorder -> obs-studio
    • Video Player -> mpv

    • Music -> Spotify

      Go over to my music if you like metal and rock
